## 200-Word Version

Use this when you need a full project description for LinkedIn, a portfolio write-up, or application material where you want to show both product thinking and technical depth.

WaveStream is an original, SoundCloud-inspired music streaming platform built as a full-stack portfolio project by Nguyễn Sơn. The goal was to create something that felt closer to a startup-grade product than a basic demo, so the project was approached as a complete system rather than a collection of isolated screens. The platform combines a Next.js App Router frontend, a modular NestJS API, PostgreSQL, Redis, MinIO, Mailpit, and Docker Compose in a pnpm monorepo structure.

On the product side, WaveStream supports public discovery, track playback, playlists, follows, likes, reposts, comments, creator uploads, owned-track management, and moderation tooling. On the engineering side, it includes refresh-cookie session bootstrapping, persistent cross-route audio playback, S3-compatible media storage, seeded demo content, automated testing, Docker smoke verification, and GitHub Actions CI/CD with GHCR image publishing.

I built WaveStream to demonstrate end-to-end ownership across UX, frontend architecture, backend design, auth, media handling, testing, and DevOps. The project reflects a product mindset as much as a coding exercise: the focus was not only on shipping features, but on closing credibility gaps, removing fake surfaces, and making the repository, deployment flow, and demo experience feel polished, runnable, and reviewable.
