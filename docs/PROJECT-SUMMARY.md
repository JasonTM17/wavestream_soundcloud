# WaveStream Project Summary

English | [Tiếng Việt](./PROJECT-SUMMARY.vi.md)

## One-Line Summary

WaveStream is a full-stack music streaming platform portfolio project built with Next.js, NestJS, PostgreSQL, Redis, MinIO, and Docker, with real playback, creator tooling, moderation flows, and CI/CD automation.

## Short Portfolio Description

WaveStream is an original, SoundCloud-inspired audio platform built by Nguyễn Sơn to demonstrate production-minded full-stack engineering. The project includes public discovery, persistent playback, creator uploads, playlist management, moderation tooling, seeded demo content, Docker-first local setup, and GitHub Actions pipelines for validation and image publishing.

## Resume Version

- Built `WaveStream`, a production-style audio streaming platform using Next.js App Router, NestJS, PostgreSQL, Redis, MinIO, and Docker.
- Implemented end-to-end auth, creator uploads, playlists, social interactions, moderation flows, and a persistent cross-route audio player.

WaveStream is a full-stack audio platform demo with real discovery, playback, uploads, playlists, moderation, Docker-first local setup, and GitHub Actions CI/CD.

## Stack Highlights

- Frontend: Next.js App Router, TypeScript, Tailwind CSS, React Query, Zustand
- Backend: NestJS, TypeORM, JWT auth, modular REST API
- Data and infrastructure: PostgreSQL, Redis, MinIO, Mailpit, Docker Compose
- Quality and delivery: Vitest, Playwright, integration testing, GitHub Actions, GHCR publishing

- A portfolio repo that is runnable, testable, and reviewable end to end
