## Short Portfolio Pitch

WaveStream is a production-minded music streaming portfolio project with public discovery, genre search, persistent playback, queue management, creator uploads, playlist workflows, reporting, admin moderation, authentication, object storage, Dockerized infrastructure, and CI/CD quality gates.

## Vietnamese Pitch

WaveStream là dự án portfolio full-stack lấy cảm hứng từ trải nghiệm chia sẻ nhạc hiện đại, tập trung vào một sản phẩm hoàn chỉnh: discovery công khai, tìm kiếm theo thể loại, player liên tục, queue, upload nhạc, playlist, báo cáo nội dung, kiểm duyệt admin, xác thực, object storage, Docker và CI/CD.

## Repository Highlights

- Real seeded audio and artwork generated into object storage for local demos.
- Full Docker stack with PostgreSQL, Redis, MinIO, Mailpit, API, and web.
- Production build, unit tests, Playwright e2e, visual snapshots, and Docker smoke checks.
- Release checklist covering secrets, storage, SMTP, database backup, smoke test, and rollback.
- Public release notes are kept in `docs/RELEASE_NOTES_v1.0.0.md`.
